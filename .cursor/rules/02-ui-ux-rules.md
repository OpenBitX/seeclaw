# React + MUI 前端开发规范

1. **严禁业务逻辑**: 前端组件中不允许出现对任务的解析、状态的持久化计算。前端只负责两件事：触发 Tauri `invoke` 和监听 Tauri `event`。
2. **MUI 深度定制**: 使用 MUI v5 的 `ThemeProvider` 统一定义赛博朋克/极客风格的主题色（高对比度，深色模式优先）。严禁在组件内写硬编码的 inline-style。
3. **状态管理**: 仅使用 Zustand 管理轻量级的 UI 状态（如：侧边栏是否展开、当前是否处于拖拽模式）。来自 Rust 的 Agent 状态流转数据直接通过统一的 Hook 注入视图。
